# 🔐 KeyVault
本地纯前端账号密码管理器，数据仅保存在本地，不上传任何隐私信息。

## 📌 项目介绍
基于 Vite + Tauri 开发，可打包为 Windows 桌面客户端。

## ✨ 核心功能
| 功能 | 说明 |
| ---- | ---- |
| ➕ 添加账号 | 填写名称、分类、网址、账号、密码、备注 |
| ✏️ 编辑账号 | 随时修改账号所有信息 |
| 🗑️ 删除账号 | 二次弹窗确认，防止误删 |
| 🔍 实时搜索 | 模糊匹配名称、账号、网址、分类、备注 |
| 🏷️ 分类管理 | 自定义分类，顶部标签快速切换 |
| 👁️ 密码显隐 | 卡片与表单均可切换密码可见/隐藏 |
| 📋 一键复制 | 账号、密码一键复制到剪贴板 |
| 🎲 随机密码 | 生成高强度随机密码，搭配强度检测 |
| 📤 导出备份 | 账号数据导出为 JSON 文件 |
| 📥 导入恢复 | 导入 JSON 备份，自动去重合并 |


## 🎨 界面特色
- 深色玻璃态设计，蓝紫渐变主题
- 卡片悬浮动画，页面流畅过渡
- 全响应式布局，适配电脑、移动端

## 🖥️ 桌面客户端
前往 [Releases](https://github.com/Kiko1125/keyvault/releases) 下载 Windows exe 安装包，双击安装即可使用。

## 📦 安装包说明
1. **标准安装版（xxx.exe）**
   常规安装程序，会写入系统开始菜单，适合长期固定使用。
2. **便携版（xxx.zip）**
   无需安装，解压即可直接运行；所有配置、数据都在程序同目录，可放U盘随身携带。

## 📌 使用说明
- 安装版：双击安装，正常启动使用
- 便携版：解压后直接运行程序，拔掉U盘数据不会留在电脑本地
所有数据仅保存在当前设备，建议定期导出JSON备份账号信息。
